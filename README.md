# Different Test Projects from Colab

Сборник исследовательских ноутбуков, объединённых в один Colab-файл. Внутри собраны эксперименты по анализу данных, машинному обучению, компьютерному зрению и финансовым временным рядам.

## Что есть внутри

Ноутбук `Test.ipynb` содержит несколько независимых блоков с разными задачами:

- регрессия и прогнозирование на табличных данных;
- эксперименты с датасетами недвижимости и Airbnb;
- задачи классификации на медицинских и других выборках;
- проекты по компьютерному зрению и распознаванию изображений;
- эксперименты с финансовыми данными и криптовалютными временными рядами.

## Технологии

- Python
- pandas, numpy, scipy
- scikit-learn
- matplotlib, seaborn
- TensorFlow / Keras
- PyTorch
- OpenCV

## Формат репозитория

- `Test.ipynb` - единый ноутбук с набором отдельных экспериментов;
- большая часть кода рассчитана на запуск в Google Colab и чтение данных с Google Drive.

## Особенности

- Это не один цельный продукт, а набор независимых исследовательских задач.
- В ноутбуке встречаются жёстко прописанные пути к датасетам на Google Drive.
- Для повторного запуска потребуется вручную восстановить соответствующие входные данные.
